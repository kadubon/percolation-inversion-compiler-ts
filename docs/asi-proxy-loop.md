# ASI-Proxy Loop

This repository treats ASI-proxy/CARA acceleration as a protocol-relative local loop, not as proof of real ASI. The loop is: finite trace, token candidate, PIC or PIC-TS checks, CCR task/residual/capital-witness import, foundry active-cut allocation, observation, and residual repair.

Safety rules: target set and baseline are declared before outcome observation; token extraction is not settlement; token admissibility is not capital admission; proxy-only evidence cannot increase safe capital; safe commands are hints, not authority; duplicate mass does not increase support.

Boundary markers for agents and CCR audits: Token extraction is not settlement. Token admissibility is not capital admission. Proxy-only evidence cannot increase safe capital.

Use the `examples/asi_proxy_loop_bundle/` files as the smallest searchable fixture set for first-time agents.
