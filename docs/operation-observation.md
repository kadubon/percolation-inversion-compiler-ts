# Operation Observation

TRC observation is window-relative. An observation report may show `dispatch_executed=true`, but physical outcome proof remains false unless a scoped verifier acceptance is present. Observation evidence does not prove hidden physical truth.

Residual classes include missing verifier, physical outcome mismatch, rollback not confirmed, hazard observation missing, provider report not normalized, incident review required, postcondition missing, resource use missing, tolerance violation, expired window, invalid schema, and profile mismatch.
